# Servernoots Public Release

This is the public release of the Servernoots cloud suite. All sensitive information has been removed. Please see the `/docs` folder for setup, configuration, and usage instructions.

## Quick Start
1. Copy any `.env.example` files to `.env` and fill in your own credentials.
2. Follow the instructions in the documentation to deploy your own instance.
3. For support, see the TROUBLESHOOTING and USAGE_GUIDE docs.
